# Discord Markdown examples

## 1) 段落

USER: 請用兩段話介紹服務特點。

BOT:
本服務以高效率與穩定性為核心，提供可靠的使用體驗。

我們重視透明與信任，會清楚說明規則與注意事項。

## 2) 標題

USER: 請用標題整理活動資訊。

BOT:
# 活動公告
## 活動時間
### 注意事項

## 3) 粗體 / 斜體 / 底線 / 刪除線

USER: 請在一句話中示範強調語法。

BOT:
**重要**事項請先閱讀。
*提醒*：逾時不補發。
__保密__內容請勿外流。
~~錯誤資訊已撤回~~

## 4) 行內程式碼

USER: 請示範指令格式。

BOT:
請輸入 `help` 以查看可用指令。

## 5) 程式碼區塊

USER: 請提供多行指令範例。

BOT:
```text
/help
/shop
/user-panel
```

## 6) 引用區塊

USER: 請引用一段提示。

BOT:
> 請先閱讀規則，再開始操作。

## 7) 列表 (僅允許 "-" 與 "數字.")

USER: 請列出注意事項與步驟。

BOT:
- 注意事項一
- 注意事項二

1. 步驟一
2. 步驟二

1. 父項目
    - 子項目

## 8) 連結 (僅允許自動連結)

USER: 請提供官方網站連結。

BOT:
<https://example.com>

## 9) 暴雷

USER: 請用暴雷遮蔽答案。

BOT:
||隱藏答案內容||

## 10) 表情符號

USER: 請示範表情符號用法。

BOT:
:smile: 歡迎加入。
😀 祝你有美好的一天。

## 11) 遊戲介紹範例

USER: 請簡要介紹骰子遊戲。

BOT:
# 遊戲介紹

## 🎲 摘星手骰子遊戲

**規則**：
- 消耗遊戲代幣擲骰子（1 代幣 = 1 顆骰子）。
- 可下注 1-10 顆骰子。
- 每顆骰子點數範圍：1-6 點。

**計分方式**：
- 總獎勵 = 點數總和 × 倍率。

**參數**：
- tokens（下注代幣數量，1-10）。
